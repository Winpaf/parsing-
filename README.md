# parsing-

## Описание
Программа парсит расписание электричек с Ярославского вокзала Москвы. Извлекает данные о рейсах электричек и позволяет фильтровать их по дням следования.

## Извлекаемые данные
- **Время отправления** (например: "04:46")
- **Маршрут** в формате "Откуда → Куда" (например: "Москва Ярославская — Фрязево")
- **Дни следования** (будни, выходные, ежедневно)

## Подготовка HTML файла

1. Перейдите на сайт: https://www.tutu.ru/station.php?mst=45807
2. Сохраните страницу как `schedule.html`:
   - Нажмите `Ctrl+S`
   - Сохраните файл в папку с программой

## Установка зависимостей

```
pip install -r requirements.txt
```

Файл `requirements.txt`:
```
beautifulsoup4
```

## Запуск программы

### Все поезда
```
python parse_sputnik.py
```

### Только будние рейсы
```
python parse_sputnik.py будни
```

### Только выходные рейсы
```
python parse_sputnik.py выходные
```

### Только ежедневные рейсы
```
python parse_sputnik.py ежедневно
```

## Пример вывода

```

05:04 - Москва Ярославская — Монино
05:10 - Москва Ярославская — Софрино
05:16 - Москва Ярославская — Фрязино Пасс.
...
```

### Пример с фильтром "ежедневно":
```
python parse_sputnik.py ежедневно
```
Вывод:
```

05:10 - Москва Ярославская — Софрино 
05:16 - Москва Ярославская — Фрязино Пасс.

```
